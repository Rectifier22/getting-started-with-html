The Web - World Wide Web
Invented in the 1990’s and currently consists of approximately 30 Billion pages. HTML is the common language of the web. While many types of files such as Plain Text, HTML, Images, Videos, and Music, HTML is the main type of document found on the web. The web is comprised of documents linked together via Hyper Links.

HTML - Hypertext Markup Language

HTML is made up of:
• Text Content
• Markup – What it looks like
• References to (e.g. images and videos)
• Links to other pages

Markup is achieved with the use of HTML "Tags."
Tags are often comprised of a set of Opening < X > and Closing tags < / X>

NOTE:Void Tags do not have a Closing Tag.

Element - Everything within a set of opening and closing tags.
Attribute - Property of the HTML element.

Inline / Block
